# TG_Bot_LM_Studio

Репозиторий содержит скрипт для запуска телеграм бота.
Для запуска необходимо предварительно установить [LM Studio](https://lmstudio.ai/) и загрузить с платформы [Hugging Face](https://huggingface.co/) языковую модель.

Также нужно предварительно создать Telegram бота и подключить к проекту.
Создать своего Telegram бота можно с помощью TG бота @BotFather.

## Установка и запуск

1. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

2. Запустите LM Studio и загрузите языковую модель. Убедитесь, что модель запущена и доступна по адресу `http://localhost:1234`.
3. Создайте файл `.env` в корне проекта и добавьте в него переменную окружения с токеном вашего Telegram бота:

   ```plaintext
   API_TOKEN=your_telegram_bot_token_here
   ```

4. Запустите скрипт:

   ```bash
   python main.py
   ```

## Функциональность бота

- Команды:
  - `/start` - выводит приветствие и список доступных команд
  - `/model` - выводит название используемой LLM
  - `/clear` - очищает контекст диалога
- Запросы пользователя пересылаются LLM, запущеной на этом компьютере, и потом ответ пересылаются пользователю

## Особенности бота

Контекст переписки сохраняеться в базе данных SQLite. При использовании команды `/clear` контекст переписки для данного пользователя очищаеться.
